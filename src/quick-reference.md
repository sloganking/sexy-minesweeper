## Quick Reference

```
WHEN A MISTAKE HAPPENS:
1. Pause, breathe
2. Assess SEVERITY → High / Medium / Low
3. Assess TYPE → Malicious? Accident? Fawning?
4. Match RESPONSE to severity
5. Focus on REPAIR, not punishment
```
