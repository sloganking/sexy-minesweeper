## Severity Scale

```
  ▲ HIGH
  │   • Permanent harm
  │   • Reputation damage
  │   • Physical injury
  │   • "Would I do this with a gun?"
  │
  │ MEDIUM
  │   • Temporary pain
  │   • Recoverable
  │   • Touch/intimacy mistakes
  │   • Emotional hurt
  │
  │ LOW
  │   • Inconvenience
  │   • Awkwardness
  │   • No real harm
  │   • Miscommunication
  ▼
```

### Simple Version

```
┌─────────────────────────────────────┐
│  HIGH    Permanent / Gun territory  │
├─────────────────────────────────────┤
│  MEDIUM  Temporary / Recoverable    │
├─────────────────────────────────────┤
│  LOW     Inconvenient / No harm     │
└─────────────────────────────────────┘
```

**Key Point:** Your response should match this scale.

                 │

        │ involved and    │      │ Any action now =    │

        ┌─────────────────┐

        └────────┬────────┘
                 │
                 ▼
        ┌─────────────────┐
        │  Is this about  │
        │  NOW or my PAST?│  ← Check your filters
        └────────┬────────┘
                 │
                 ▼
        ┌─────────────────┐

### Simplified Steps

1. PAUSE
2. VERIFIED? (If NO → STOP)
3. SEVERITY? (Low / Medium / High)
4. TYPE? (Malicious / Accident / Fawning)
5. NOW or PAST?
6. MATCH RESPONSE
7. AIM for REPAIR

## Drama Triangle

### The Problem (Dysfunctional)

```
                    PERSECUTOR
                   "It's your fault"
                        ▲
                       / \
                      /   \
                     /     \
                    /       \
                   /         \
           VICTIM ───────────── RESCUER
         "Poor me"            "Let me save you"

              ↻ ROLES ROTATE ↺
```

### The Solution (Healthy)

```
                    CHALLENGER
                  "I'll push you to grow"
                        ▲
                       / \
                      /   \
                     /     \
                    /       \
                   /         \
          CREATOR ───────────── COACH
       "What do I want?"    "How can I support?"

              ✓ STABLE ROLES ✓
```

## Quick Reference Card

```
WHEN A MISTAKE HAPPENS:
1. Pause, breathe
2. Have I VERIFIED with primary sources? (If NO → STOP)
3. Assess SEVERITY → High / Medium / Low
4. Assess TYPE → Malicious? Accident? Fawning?
5. Check: Is this NOW or my PAST?
6. Match RESPONSE to severity
7. Focus on REPAIR, not punishment

REMEMBER:
• 9/10 times it's a good-faith mistake, not malice
• Your response should match severity, not your fear
• If you haven't verified, your opinion is unqualified
• Everyone has responsibility—including you
```
